## Calculation Formula Labels

### Step-by-Step Display

1. **Addition Symbol**: `+`
2. **Subtraction Symbol**: `-` or `–`
3. **Multiplication Symbol**: `×`
4. **Division Symbol**: `/` or `÷`

### Example Labels (as shown in result):

```
Format: [Value] [Operator] [Value] [Label]

English Examples:
- "2500 + 42 Me.Touch"
- "5952 - 5000 Ga. Topna"
- "952 × 2.75 Co.Touch"

Gujarati Examples:
- "૨૫૦૦ + ૪૨ મે.ટચ"
- "૫૯૫૨ - ૫૦૦૦ ગા.ટોચના"
- "૯૫૨ × ૨.૭૫ કો.ટચ"
```

## Number Formatting

### English Format
- Decimal separator: `.` (period)
- Example: `1014.50`

### Gujarati Format
- Uses same decimal format as English (for consistency)
- Gujarati numerals can be added if needed:
  - 0-9 → ૦-૯

## Implementation Notes

1. All translations are centralized in `LanguageService`
2. Language preference is saved in SharedPreferences
3. UI updates automatically when language changes
4. No need for app restart
5. All numeric values remain in English numerals for calculation purposes
